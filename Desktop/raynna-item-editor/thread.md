# Raynna Item Editor

## Raynna Item Editor Download

A desktop item editor for cache-based item work, focused on fast previewing and direct editing without digging through raw definitions by hand.

This editor is built for practical item work:

- edit item definitions
- edit client params
- edit ground options and inventory options
- edit recolors with live preview
- edit face textures
- preview inventory models
- preview male and female wear models
- pack changes back into cache

## Main Features

- Live item preview with zoom, rotation, and offset controls
- Inventory, male, and female preview modes
- Recolor editor with direct color-code editing
- Face highlighting for recolor work
- Face texture list with texture picker
- Structured item param editor with add, edit, and delete
- Ground and inventory option editor with presets and custom text
- Direct cache packing from the editor

## What You Can Edit

- Name
- Price
- Stackable flag
- Members flag
- Model id
- Model zoom
- Model rotations
- Model offsets
- Equip slot
- Equip type
- Male equip models
- Female equip models
- Male wear offsets
- Female wear offsets
- Note and lend links
- Team id
- Client params
- Ground options
- Inventory options
- Model recolors
- Face textures

## Workflow

1. Load your cache
2. Select an item from the list
3. Edit definition values, params, options, recolors, or textures
4. Use the live preview to inspect changes
5. Pack the item back into the cache

## Recolor Tools

- Click a recolor entry to edit it
- Copy the original color code directly
- Paste a new color code directly
- Use sliders for hue, saturation, and lightness
- Click the preview model to pick faces for recolor work
- Toggle face highlighting from the recolor section

## Texture Tools

- Browse model face textures in a face list
- Apply textures from the texture picker
- Clear selected face textures
- Preview textured inventory models

## Option Editing

- Ground options and inventory options are shown as slot-based lists
- Use presets for common actions
- Apply custom action text directly
- Edit or clear individual slots

## Item Params

- Right-click param list entries for edit and delete
- Add new params
- Edit existing params
- Apply raw param text if needed

## Notes

- Inventory preview is the most complete path for texture editing work
- Male and female previews are useful for worn-model inspection
- Some cache/model edge cases may still depend on the specific revision being edited
